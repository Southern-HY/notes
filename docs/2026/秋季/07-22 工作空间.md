# 工作空间

*7.22 周三*

进入了数字化时代之后，人们的“工作空间”和以前相比变小了。

现在的工作空间，指的是“数字空间”，即人们使用各种软件对数据进行加工、处理的空间。

比如手机，虽然非常便携，触屏这种操作方式也很迅速，但是由于只有手掌大的屏幕大小，工作空间被限制（开一个以上的APP就会十分拥挤），导致做大部分工作都很不方便，除了少数如直播等工作，或者那些工作内容实际大部分都还在现实中、只用手机作辅助的工作，比如送快递。

平板电脑稍微好些：拥有一本书大小的屏幕，可以自然地显示文档、书籍、播放媒体、进行视频通话，而且还能用电容笔进行随心写画。在平板电脑上，人们就能更加方便地进行传统的各种工作。然而，一个平板同时只能以自然大小显示一份文档，这和我们在传统物理工作空间中自由摆放、同时查阅多份文档的能力相比，还是略显拘束。

笔记本和台式电脑虽然屏幕更大，而且有专门设计的键鼠操作方式，但仍然不够方便。首先，鼠标在每次点击前，需要有一个移动的过程，无法做到指哪打哪，这会耗费掉操作者不少的精力。其次，当我们同时开几个有相互重叠的窗口，并在之间反复切换时，仍然会觉得不方便。

不过，VR眼镜和脑机接口（如果有可能的话）会彻底解放我们的数字工作空间。在虚拟空间中，我们可以拥有一个根据自己个人喜好定制的小房间；在柜子、桌子、壁炉、窗台或墙上，各种文档、窗口想怎么放就怎么放，不用再受到任何拘束。同时，眼球跟踪或脑电波空间信号读取能技术让我们瞬间就定位到想要指向的地方，使得我们在空间内的各种操作变得更加灵活、方便。
